# PicGo+Typora强强联手

#### 下载PicGo

[下载地址]: https://github.com/Molunerfinn/PicGo/releases

#### 关联Github仓库

1. 创建一个github仓库
2. 个人中心--setting--Developer settings--Personal access tokens
3. 点击右上角的Generate new token创建一个token
4. 选择repo，使这个token可以操作Github仓库
5. 复制token
